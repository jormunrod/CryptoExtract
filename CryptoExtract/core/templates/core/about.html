About CryptoExtract

Learn more about our project and how it works.

What is CryptoExtract?

CryptoExtract is a web application designed as a final project for the course Acceso Inteligente a la Información. It provides a platform to scrape, index, and analyze cryptocurrency data from Yahoo Finance.

How does it work?

The project works by fetching real-time cryptocurrency data using web scraping techniques with BeautifulSoup. The data is then indexed using Whoosh, a fast and efficient search engine library for Python.

Features

  • Real-time data scraping from Yahoo Finance.
  • Efficient data indexing and search functionality.
  • Responsive and user-friendly interface.

Developed By

This project was developed by Jorge Muñoz Rodríguez as part of a university assignment.

About CryptoExtract

Learn more about our project and how it works.

What is CryptoExtract?

CryptoExtract is a web application designed as a final project for the course Acceso Inteligente a la Información. It provides a platform to scrape, index, and analyze cryptocurrency data from Yahoo Finance.

How does it work?

The project uses BeautifulSoup for web scraping to fetch real-time cryptocurrency data. The data is then indexed using Whoosh, allowing for fast and efficient querying and analysis. Users can interact with the data through various views and tools available on the platform.

Features

The application includes the following functionalities:

  • Home Page: Provides an overview of the project and displays the last time the data was updated.
  • Load Data: Fetches real-time cryptocurrency data from Yahoo Finance and overwrites the existing data.
  • View Data: Displays all the indexed cryptocurrencies in a responsive table with sortable and scrollable features.
  • Search Cryptocurrencies: Enables users to search for cryptocurrencies by name using a robust search engine.
  • Compare Cryptocurrencies: Enables users to compare two cryptocurrencies side by side, showing metrics like price, market cap, and more.
  • Price Range Filter: Allows users to filter cryptocurrencies by specifying a price range.
  • Top 5 by Percentage Change: Displays the top 5 cryptocurrencies with the highest percentage change in the last 24 hours.
  • Market Cap Chart: Displays a bar chart comparing the market capitalization of cryptocurrencies, providing a clear visualization of their relative sizes.
  • Scatter Plot (Price vs. Volume): Visualizes the relationship between the price and trading volume of cryptocurrencies using an interactive chart.

Developed By

This project was developed by Jorge Muñoz Rodríguez as part of a university assignment for the course Acceso Inteligente a la Información. It demonstrates the integration of web scraping, data indexing, and visualization tools in a cohesive web application.
